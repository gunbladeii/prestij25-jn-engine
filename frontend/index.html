INFO SISTEM
Tentang Sistem

MoE AI-Complaint — Sistem Imbangan Semak Pintar. Supreme Truth & Audit Node dalam ekosistem matrix 25×25.

Seni Bina
Ejen A — Semantic Ingestion & Mapping
NER + Weighted Taxonomy Scoring
Ejen B — Cross-Examination & Anomaly
DI Computation + 8-Flag Generator
Ejen C — Executive Briefing
Policy Matrix + Directive Generator
Tindanan Teknologi
Backend — FastAPI + Python 3.12
Database — PostgreSQL 16 (JSONB)
Frontend — Vanilla HTML/CSS/JS
Deploy — Docker / Netlify / Render
Endpoint API
POST/api/v1/matrix/ingest— Ingest payload dari sistem matrix
GET/api/v1/matrix/executive-brief/{case_id}— Retrieve executive brief
GET/api/v1/matrix/cases— List all processed cases
GET/api/v1/health— Engine health check
GET/api/docs— Interactive API docs (Swagger)
